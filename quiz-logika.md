## 1. Quiz 1 \(Guide\)

> **Menghasilkan bilangan Fibonachi dengan inputan N**

* ### Function

  ```java
  public int[] getFibonachi(int n) {
  	int[] data = new int[n];
  	data[0] = 1;
  	data[1] = 1;
  	for(int i = 2; i < n; i++) {
  		data[i] = data[i - 1] + data[i - 2];
  	}
	
  	return data;
  }

  ```
